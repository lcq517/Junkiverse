各页面功能
1

首页 / 创作入口

启动页，展示产品介绍和"开始创作"按钮
产品介绍 开始创作按钮 最近作品（可选）
2

拍照 / 上传页

调用相机拍照或从相册选图，自动抠图并预览结果
拍照 / 相册选择 AI 自动抠图 抠图结果预览 重拍 / 确认
3

拼贴编辑器

自由画布，把抠好的部件拖动、缩放、旋转，组合成角色
Canvas 自由画布 拖拽 / 缩放 / 旋转 添加新部件 删除 / 图层调整 生成角色按钮
4

生成加载页

AI 生成中的等待动画，给用户预期
加载动画 进度提示 趣味文案
5

结果展示页

展示 AI 生成的角色立绘和档案故事，支持保存和分享
角色立绘展示 档案故事卡片 保存图片 分享链接 再来一个
后端 API（MVP）
方法 接口 说明
POST /api/generate/character 提交拼贴，生成角色立绘+故事
GET /api/generate/{id} 查询生成结果（轮询用）
POST /api/works 保存作品到服务器（可选）
GET /api/works/{id} 获取单个作品详情
核心数据模型
Work { // 作品   id: Long,   collageImage: String, // 拼贴图 URL   characterImage: String, // AI 生成立绘 URL   characterName: String, // 角色名   characterStory: String, // 档案故事   status: Enum, // GENERATING / DONE / FAILED   createdAt: DateTime }
